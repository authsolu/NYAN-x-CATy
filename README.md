NutzWk
======

基于Nutz的开源企业级开发框架


文件编码全部为UTF-8,可以导入Eclispe、IDEA中,jdk7,tomcat 6/7.
需执行nit_mysql.sql进行数据库初始化

本框架的代码生成器（兼容mysql、oracle等数据库）：
https://github.com/Wizzercn/NutzCodematic


本框架已成功应用于XX省交通厅网络问政平台、XX省交通厅CMS内容管理系统、XX公司舆情监测管理中心等项目。

======
